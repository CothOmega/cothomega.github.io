Fixed Positioning

An element with fixed position is positioned relative to the browser window. It will not move even if the window is scrolled. It is also important to note that items with a fixed position are removed from the normal flow of the HTML. Other elements behave as though the fixed position element does not exist. And fixed positioned elements can overlap other elements.

Up in the top right corner, you may notice some fixed position text. If you were to scroll this webpage (you have have to vertically shrink the page since there isn't much content), that content will not move. It will stay in its location. You have have visited websites that have banners across the top that don't move as you scroll. Another good example!

Here is the syntax of that fixed position text:
p.pos_fixed {
position: fixed;
background-color: white;
color: red;
top: 15px;
right: 40px;
padding: 1px;
}

Then you just have to call the pos_fixed class on whatever p-tag you would like to fixate!

Relative Positioning

An element with relative position is positioned according to where its normal location would have been.

For example, I used relative positioning to move this p-tag 15px to the right of where it would have sat normally.

The CSS to do that is:
p.pos_right{
position: relative;
left: 15px;
background-color: white;
color: red;
padding: 1px;
}

And like before, just call the pos_right class on whatever p-tags you would like to shift.

As well as shifting left and right, you can also position elements in reference to top and bottom. But we must keep in mind that The content of relatively positioned elements can be moved and overlap other elements, but the reserved space for the element is still preserved in the normal flow. So if I move an element UP, the line where that text would have NORMALLY been will still be there as if the text was.

Absolute Positioning

Absolute positioned elements are a mixed breed from Fixed and Relative. Like Fixed, they can be placed anywhere you would like on the page, but they are not fixed to that location relative to your browser window. And the document and other elements treat an absolute elements like it does not exist, and in that way it does not interrupt the normal flow of a page like Relative positioned items do.

For this example, I chose to absolutely position my h2 tag left 100px and 70px from the top.

h2{
position: absolute;
left: 100px;
top: 70px;
background-color: white;
color: red;
padding: 2px;
}
